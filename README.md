# Air Quality Analysis using Artificial Intelligence

This project focuses on analyzing air quality data and using an AI model to predict carbon monoxide (CO) levels based on environmental variables such as temperature, humidity, and other air pollutants.

## Project Objective
The objective of this project is to investigate whether artificial intelligence can accurately predict CO levels using environmental data. This study demonstrates how machine learning techniques can be applied to real-world air quality monitoring problems.

## Dataset
The dataset used in this project is the **Air Quality Dataset** from the UCI Machine Learning Repository.

It contains real air quality measurements collected hourly from a gas multisensor device installed in an Italian city between March 2004 and February 2005.

The dataset includes:
- CO concentration (target variable)
- Temperature
- Relative Humidity
- NO2 concentration
- NOx concentration
- Benzene concentration
- Sensor measurements

## Methodology
The following steps were performed in this project:

1. Data loading using Google Colab
2. Data cleaning and preprocessing
3. Feature selection
4. Model training using Linear Regression
5. CO level prediction
6. Model evaluation using graphs and performance metrics

## Tools and Technologies
- Python
- Google Colab
- Pandas
- Scikit-learn
- Matplotlib
- Machine Learning (Linear Regression)

## Results
The AI model was able to predict CO levels based on environmental variables. The results were evaluated using prediction graphs such as Actual vs Predicted values and Prediction Error Graphs.

## Web Application
A simple web interface was also created to allow users to input environmental variables and predict CO levels using the trained model.
